Detailed Description

Reads from the kernel character driver file at /dev/CDS to catch coin time signatures. Signatures are read as comma seperated values, then they are extracted and sent to a CoinIdentifier. The class inherits from OSAPI ThreadFunctor class to enable starting the c++ class as a thread using POSIX-threads.
