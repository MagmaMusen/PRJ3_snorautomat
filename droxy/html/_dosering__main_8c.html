◆ main()

int main ( )

The program will run in the main. The InitializeSystem will be called to initialize the I2C function and PWM function. The for Loop contains the Dosering function which is the most important part in the program. This the if() will first check on the write slave flag, if it is low nothing will happen but the function will be runned agin. the function will turn on the specific servo motor all after which is the buffer size contains, and after that will generate the PWM signal to run the Motor. when the motor is done running, the slave buffer and flag will be cleared.
