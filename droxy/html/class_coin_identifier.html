Detailed Description

Identifies coin values based on received coin time signatures and sends the value to specifiable receiver. Coin width boundaries and sensor distances have been hard-coded into the class. The class inherits from OSAPI ThreadFunctor class to enable starting the c++ class as a thread using POSIX-threads.
